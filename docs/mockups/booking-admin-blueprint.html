Placement rules
  • `Book Session` stays top-right and primary.
  • `Manage Bookings` stays top-right for list-wide admin editing.
  • `Manage Clients` stays top-right because clients are part of the same surface.
  • Critical session actions move to each booking row: `Edit`, `Assign`, `Cancel`.
  • `Needs Attention` gets its own box so problems are visible before scrolling the board.
  • Member bubbles belong in a contained strip, not floating loose.
