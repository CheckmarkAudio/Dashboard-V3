# Social Post Standard

## Purpose

Use this file as the base structure for reusable social media captions and posts.

## Use When

- Creating a post for LinkedIn, Instagram, Facebook, or another public channel.
- Turning a past post into a reusable template.
- Preparing social copy for workspace portal reuse.

## Do Not Use When

- The post requires platform-specific formatting that should live in a dedicated file.
- The message includes private client context.

## Template

[Strong opening line.]

[Short supporting context.]

[Clear takeaway, offer, or update.]

[Primary CTA]

[Optional hashtags]

## Customization Notes

- Keep the opening line specific and useful.
- Replace broad claims with concrete details.
- Adapt CTA language to the platform.
- Use hashtags sparingly.

## Review Checklist

- First line can stand alone.
- CTA is clear.
- No stale dates or private details remain.
- Platform-specific formatting has been checked.
